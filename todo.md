صفحة الموكلين
نحذف اسم العائلة ؛ يكفي فقط : اسم الموكل ( تكفينا خانة واحدة هي: اسم الموكل )

صفحة القضايا:  
نوع القضية: المدني، الإجتماعي، العقاري، شؤون الأسرة، التجاري، البحري، الاستعجالي، الجنح، المخالفات، الأحداث.
ملاحظة: قسم تطبيق العقوبات يوجد على مستوى محكمة مقر المجلس فقط ( مثال: مجلس قضاء البويرة. --- قسم تطبيق العقوبات يكون في محكمة البويرة فقط ) ، باقي الولايات كيف كيف إلا العاصمة يوجد في محكمة سيدي أمحمد )
المحكمة ونوع المحكمة = خطأ ؛ الصحيح هو نوع القضاء ( القضاء العادي ، القضاء الإداري، القضاء التجاري المتخصص ) ... اتبع الجدول الذي ارسلته لك ...
دور الموكل = نغيرها إلى: صفة الموكل
القاضي = تحذف هاته الخانة
الطرف المقابل = نغيرها إلى الخصم \
محامي الطرف المقابل = نغيرها إلى محامي الخصم
الحالة = على مستوى الدرجة الأولى، في إطار التسوية، مغلقة، في الاستئناف، طعن غير عادي.
الأولوية: عادي ، قضاء استعجالي
تاريخ البداية + تاريخ النهاية = خطأ تحذف ؛ نضع تاريخ التكليف فقط.
المبلغ المطالب به = نغيرها إلى: الأتعاب المتفق عليها

صفحة إدارة الجلسات
جميع القضايا ( الأنواع خطأ ) = فيها: الشق جزائي ، الشق مدني ، الشق الإداري
جميع الحالات: تحقيق، في التقرير، في المرافعة، لجواب الخصم، لجوابنا، في المداولة، مؤجلة، جلسة المحاكمة
عند الضغط على إضافة جلسة جديدة:
نوع الجلسة: المدني، الإجتماعي، العقاري، شؤون الأسرة، التجاري، البحري، الاستعجالي، الجنح، المخالفات، الأحداث.
المحكمة = الجهة القضائية
القاضي = قاضي الجلسة
الحالة = للتكليف، مجدولة، مؤجلة ، لحضور الأطراف

صفحة إدارة المستندات
ممتازة
نزيل فقط ؛ جميع الأنواع ( نخلوها عامة بلا أي تحديد أفضل )

صفحة إدارة الفواتير ( نغيرها إلى إدارة الأتعاب والمداخيل )
جميع الحالات = مدفوعة، مدفوعة جزئيا، متأخرة
إضافة فاتورة جديدة ( نغيرها إلى تلقي أتعاب ):
رقم الفاتورة = يكون اوتوماتيكي ( سهلها تسهال )
القضية ( اختياري ) = اجباري ( لضبط المداخيل )
تلريخ الفاتورة = يكون اوتوماتيكي
الضريبة = تحذف لا داعي لها

صفحة إدارة المواعيد
في الجدول : ممتاز ، نحذف جميع الحالات فقط
